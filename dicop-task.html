Política de Privacidade

DICOP TASK
Última atualização: 21/04/2025

1. Coleta de Dados

Não coletamos dados pessoais.

  • O aplicativo não solicita, armazena ou compartilha informações pessoais, como nome, e-mail, localização ou qualquer outro dado identificável.
  • As tarefas criadas pelo usuário são armazenadas exclusivamente no dispositivo e não são acessíveis ou transmitidas a terceiros.

2. Armazenamento Local

  • Todos os dados inseridos (tarefas, notas, etc.) são salvos localmente no dispositivo do usuário.
  • Não temos acesso a essas informações, e sua segurança é de responsabilidade exclusiva do usuário.

3. Uso de Dados

  • O aplicativo não utiliza os dados das tarefas para fins publicitários, análises ou qualquer outro uso externo.
  • Não há integração com serviços de terceiros (como redes sociais, anúncios ou analytics).

4. Responsabilidade do Usuário

  • O desenvolvedor não é responsável por:
    • Perda de dados devido a falhas técnicas, exclusão acidental ou danos ao dispositivo.
    • Acesso não autorizado ao dispositivo do usuário.

5. Atualizações e Mudanças

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por meio do aplicativo ou de notificações na Google Play.

6. Contato

Para dúvidas sobre privacidade, entre em contato em: contato@dicop.com.br.

Notas Importantes:

  • Esta política segue os requisitos mínimos da Google Play para apps que não coletam dados.
  • Se futuramente o app incluir funcionalidades que coletam dados, esta política deverá ser revisada.

Ao utilizar o aplicativo, você concorda com os termos desta política.
