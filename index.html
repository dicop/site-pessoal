Carreira

Trabalho com desenvolvimento de sistemas há mais de 14 anos, comecei desenvolvendo pequenas aplicações web usando PHP e após dois anos passei a desenvolver em Java, usando tecnologias como JSP, JDBC, Servlet, CSS, Javascript e HTML.

Com o passar do tempo fui me aprofundando no mundo Java, trabalhando com diversos frameworks, como por exemplo: Spring, Struts, JPA, EJB, JSF, JAX-RS, JAX-WS, iReport, entre outros. Dentro das principais ferramentas e tecnologias utilizadas estão: Banco de dados Oracle, Servidor de Aplicação Weblogic, Jenkins e Git.

Em minha carreira como desenvolvedor passei também a desenvolver aplicativos para celulares baseados em Android, utilizando Java. Há 4 anos atrás, com a ascenção de tecnologias hibridas, passei a trabalhar também com Flutter, linguagem multiplataforma que permite desenvolver aplicativos para Android e IOS.

Participei/participo do desenvolvimento de sistemas dos mais variados tipos, integrações com órgãos fiscais (NF-e, CT-e e MDF-e), integrações com outros sistemas baseados em webservices (SOAP/ REST), sistemas corporativos tradicionais, ferramenta para geração de relatórios / indicadores de BI, painéis de acompanhamento de processos internos, aplicativos nativos android e aplicativos multiplataforma em flutter.

Durante todo esse período me formei em sistemas de informação, fiz duas certificações Java e diversos cursos voltados para Java e desenvolvimento Mobile (Android / Flutter).

Atualmente as principais tecnologias que utilizo são: SQL, JPA, EJB, JSF (Primefaces), iReport, JAX-RS, Android (Java) e Flutter.
